# 雪場 Logo 收集指南

本目錄用於存放所有 43 個日本滑雪場的官方 Logo 圖片。

## 📋 待收集 Logo 清單

共需收集 **43 個雪場** 的官方 Logo。

### 收集方法

由於所有雪場官網都有 Cloudflare 等防爬蟲保護，自動化下載失敗，需要**手動收集**。

#### 方法 1：瀏覽器開發者工具（推薦）
1. 訪問雪場官方網站
2. 按 `F12` 打開開發者工具
3. 切換到 `Network` 標籤
4. 過濾器選擇 `Img`
5. 重新整理頁面
6. 找到網站 Logo 圖片（通常在 header 區域）
7. 右鍵點擊圖片 → `Open in new tab` → 另存為

#### 方法 2：查看網頁原始碼
1. 訪問雪場官方網站
2. 右鍵 → `檢視網頁原始碼`
3. 搜尋關鍵字：`logo`、`og:image`、`twitter:image`
4. 找到 Logo 圖片 URL
5. 在新分頁開啟 URL → 另存為

#### 方法 3：社交媒體資料圖片
1. 在 Instagram/Facebook 搜尋雪場官方帳號
2. 下載其頭像或封面圖（通常是官方 Logo）
3. 注意：Instagram 頭像通常較小，優先使用官網 Logo

#### 方法 4：品牌資源網站
- 訪問 [Brandfetch](https://brandfetch.com/) 搜尋雪場名稱
- 部分知名雪場可能有官方品牌資源包

## 📐 格式要求

### 檔案命名
- **格式：** `{resort_id}.png` 或 `{resort_id}.jpg` 或 `{resort_id}.svg`
- **範例：** `hokkaido_rusutsu.png`、`hakuba_happo_one.png`

### 圖片規格
- **最小尺寸：** 200 x 100 像素
- **推薦尺寸：** 400 x 200 像素或更大
- **格式優先順序：**
  1. **PNG** - 透明背景（最佳）
  2. **SVG** - 向量圖（最佳，但需確保相容性）
  3. **JPG** - 有背景（次選）

### 品質要求
- 清晰、高解析度
- 官方發布的 Logo（不是網站截圖）
- 盡可能使用透明背景版本

- **優先收集熱門雪場：** Rusutsu、Niseko、Hakuba、GALA Yuzawa、Naeba

## 🆘 需要協助？

1. 檢查官網是否有「媒體中心」或「品牌資源」頁面

3. 使用 Google 圖片搜尋：`{雪場名稱} logo`
4. 查看雪場的 Facebook/Instagram 官方帳號頭像
