# 🎿 滑雪助手使用指南

## 快速上手

滑雪助手是你的智能行程規劃夥伴！只要用自然語言跟我聊天，就能輕鬆建立和管理滑雪行程。

---

## 📝 基本使用

### 建立行程

只要告訴我雪場、日期和天數，我會自動幫你建立行程：

```

```

```
白馬明天開始3天
```

```
二世谷下週五出發，停留5天
```

### 查看行程

```
查看我的行程
```

```
我的行程列表
```

## 🎯 進階功能

### 1. 公開行程找雪伴

想找滑雪夥伴？加上「公開」或「找伴」關鍵字：

**關鍵字**：
- `公開` - 設定為公開行程
- `開放` - 開放給其他人申請
- `找伴` / `找雪伴` - 尋找雪伴
- `徵伴` - 徵求雪伴
- `揪團` - 揪團一起去

**範例**：
```
野澤12月3-8並公開
```

```
白馬12月找伴
```

```
二世谷開放徵伴
```

### 2. 指定雪伴人數

可以告訴我你想找幾個人：

**關鍵字**：
- `找3人` - 找3個雪伴
- `徵5位` - 徵求5位雪伴
- `要4人一起` - 想要4個人一起
- `最多3人` - 最多3個人

**範例**：
```
野澤12月公開找3人
```

```
白馬12月徵5位雪伴
```

```
二世谷開放，要4人一起
```

### 3. 完整範例

結合所有功能：

```
新增野澤12月3-8並公開徵5位雪伴
```

助手會回覆：
```
好的！正在建立行程：
📍 雪場：野澤溫泉滑雪場
📅 日期：12/3 - 12/8
⏱️ 天數：5 天
👥 公開行程（找 5 人）
```

## 💡 使用技巧

### ✅ 推薦做法

1. **自然對話**：像跟朋友聊天一樣，不用刻意使用特定格式
   ```
   我想去野澤12月3號開始滑5天，公開找3個人一起
   ```

2. **分段提供資訊**：如果一次說不完，可以分次補充
   ```
   你：野澤12月3-8
   助手：還需要確認這是公開還是私密行程嗎？
   你：公開找3人
   ```

3. **修改行程**：可以隨時修改
   ```
   改成白馬
   ```

### ❌ 常見問題

1. **日期不明確**
   - ✗ 「12月去野澤」（沒有具體日期）
   - ✓ 「12月3日去野澤」

2. **雪場名稱錯誤**
   - ✗ 「長野滑雪」（太籠統）
   - ✓ 「野澤」或「白馬」（具體雪場名稱）

## 🔗 相關功能

建立公開行程後，可以：

1. **雪伴公佈欄**：在公佈欄查看所有公開行程
2. **週過濾器**：按週篩選行程，避免資訊過載
3. **申請加入**：申請加入其他人的公開行程
4. **管理申請**：查看和回應其他人的申請
